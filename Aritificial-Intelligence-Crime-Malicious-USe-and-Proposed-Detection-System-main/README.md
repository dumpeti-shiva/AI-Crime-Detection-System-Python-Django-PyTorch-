# AI Crime Detection System  
*A Deep Learning–based Crime Detection Web Application*

## 📌 Project Overview
The **AI Crime Detection System** is a web-based application designed to detect and classify suspicious or criminal activities from images or video streams using **Deep Learning**.  
It leverages **PyTorch** for model training and inference and **Django** for backend web integration.

The system aims to assist law enforcement and security agencies by providing **automated crime detection** using AI-powered computer vision techniques.

---

## 🚀 Features
- 🔍 Detects suspicious/criminal activities using deep learning
- 🧠 AI model built with **PyTorch**
- 🌐 Web application using **Django**
- 📷 Image/Video input support
- 📊 Prediction results displayed on the web interface
- 🛡️ Useful for surveillance and security systems

---

## 🛠️ Tech Stack
- **Programming Language:** Python  
- **Deep Learning Framework:** PyTorch  
- **Web Framework:** Django  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** SQLite (default Django DB)  
- **Libraries:** OpenCV, NumPy, TorchVision
